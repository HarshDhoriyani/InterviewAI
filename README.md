# InterviewAI 🚀

InterviewAI is an intelligent mock interview platform designed to help users prepare for technical and behavioral interviews using AI-driven interactions. The platform simulates real interview scenarios, evaluates responses, and provides actionable feedback to improve performance.

---

## ✨ Features

* 🎯 AI-powered mock interviews
* 💬 Real-time question & answer interaction
* 📊 Performance analysis and feedback
* 👤 User profile with interview history
* 🔐 Authentication (login/signup)
* 🧠 Domain-specific interview sessions (DSA, HR, etc.)

---

## 🛠️ Tech Stack

### Frontend

* React.js / Next.js
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Other Tools

* OpenAI API (for AI responses)
* JWT Authentication

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/interviewai.git
cd interviewai
````

### 2. Install dependencies

#### Frontend

```bash
cd client
npm install
```

#### Backend

```bash
cd server
npm install
```

### 3. Environment Variables

Create a `.env` file in the server folder and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_api_key
```

---

## ▶️ Running the Application

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```
